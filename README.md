Projeto Fábrica de Confecção de Vestuários
Este projeto foi desenvolvido de forma rápida e está em andamento, ainda incompleto. O código atual ainda precisa de melhorias para ficar mais limpo, organizado e profissional.

O objetivo principal deste projeto é criar uma solução para armazenar informações sobre todas as máquinas utilizadas em uma fábrica de confecção, incluindo os diferentes layouts de máquinas utilizados ao longo do tempo. Isso é importante porque, frequentemente, é necessário alterar o layout das máquinas devido a defeitos ou à falta de operadores. Além disso, o sistema registra quanto cada máquina produziu em um determinado layout, por dia.

O desenvolvimento foi feito às pressas, e o projeto ainda está em fase inicial. Ainda falta deixar o código mais limpo e organizado. O principal motivo pelo qual ainda não investi na organização e modernização do código foi a decisão da empresa para a qual estava desenvolvendo o projeto. Apesar de mostrar aos poucos a evolução e os benefícios de uma estrutura digital mais moderna, eles insistiram em não ver a necessidade de uma organização mais eficiente e atualizada naquele momento.

Este projeto serve como uma base inicial e um aprendizado ao longo do caminho. Planejo continuar aprimorando, deixando o código mais limpo e estruturado assim que possível.
